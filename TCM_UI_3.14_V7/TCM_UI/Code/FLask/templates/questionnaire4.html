中医体质分类与判定自测表

答案没有对错之分，大家选择符合自己实际情况的答案即可。

计分方法与评判标准：

每条题目下设5级答案，由无到有的倾向性给出1~5分的分值（其中标有*的条目为逆向计分项目)，以单选方式选择，然后对每类的原始分采用简单求和法。
原始分数 = 各条题目分支的累加和。
转化分数 = [ ( 原始分 - 条目数 ) / ( 条目数×4 ) ] × 100。
